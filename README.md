# My Project

## Main technology

- react
- redux-toolkit, redux thunk
- react router dom
- axios
- ant design

## Install Server

- Move path

```shell
$ cd fakeAPI
```

- Install package dependencies

```shell
$ yarn install # npm install
```

- Run server

```shell
$ yarn start # npm run start
```

## Install App

- Clone the project

```shell
$ git clone git@github.com:tungkemm/myproject.git
```

- Install package dependencies

```shell
$ yarn install # npm install
```

- Run app

```shell
$ yarn start # npm run start
```
