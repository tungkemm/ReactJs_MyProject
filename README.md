# My Project

## Main technology

- reactjs
- redux-toolkit, redux thunk
- react router dom
- axios (fakeapi by jsonserver)
- ant design, css module

## Clone App

- Clone the project

```shell
$ git clone git@github.com:tungkemm/myproject.git
```

## Install JsonServer

- Move path

```shell
$ cd fakeAPI
```

- Install package dependencies

```shell
$ yarn install # npm install
```

- Run server

```shell
$ yarn start # npm run start
```
